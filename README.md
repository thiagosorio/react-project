# Calculadora de IMC - React + Vite

Aplicação web para calcular **IMC (Índice de Massa Corporal)** a partir de **altura** e **peso**, exibindo o resultado e a **classificação** conforme a referência padrão (OMS), com destaque automático da faixa correspondente na tabela.

## Funcionalidades

- Entrada de **altura (m)** e **peso (kg)**
- Cálculo do **IMC** via botão
- Exibição de **IMC** e **classificação**
- Tabela de faixas com **destaque da classificação atual**
- Layout **responsivo** (mobile e desktop)

## Tecnologias

- React
- Vite
- JavaScript (ES6+)
- CSS Modules

## Como executar

```bash
npm install
npm run dev
```

## Build de produção (opcional)

```bash
npm run build
npm run preview
```

## Notas de implementação

- Componentização em React e estado com Hooks
- Estilos com **CSS Modules** (escopo por componente)
- Layout com foco em clareza visual e usabilidade
